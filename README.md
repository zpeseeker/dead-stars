# Dead Inventors, Vanishing Scientists & The Energy They Knew

A static timeline of researchers, engineers, and officials in plasma, fusion, aerospace, and related fields—documented events, sources, and context from the late 19th century to the present.

**Live site (GitHub Pages):** [https://zpe.quest](https://zpe.quest)

## Repository

Source for the site: [https://github.com/zpeseeker/dead-stars](https://github.com/zpeseeker/dead-stars)

The site is a single-page HTML document (`index.html`) with embedded styles and scripts—no build step required.
